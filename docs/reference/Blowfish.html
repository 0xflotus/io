Blowfish
module Blowfish
category Encryption
description The Blowfish object can be used to do encryption and decryption using the blowfish algorithm. Example use; 
	
bf = Blowfish clone
bf setKey("secret")
bf beginProcessing
bf inputBuffer appendSeq("this is a message")
bf process
bf endProcess
bf outputBuffer // this contains the encrypted data
