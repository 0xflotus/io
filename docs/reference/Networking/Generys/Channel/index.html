Channel Proto







Channel provides seamless integration of FutureResponse and WebSocket implementations.
 
 
 


send(message, excludeList)

Send message to all subscribers. Returns message.

excludeList can contain list of subscribers (their name properties) to which message won't be sent.

streamFor(subscriberName)

Returns FutureResponse or WebSocket with provided name
subscribe(subscriber)

Adds subscriber to the channel. Subscriber can be FutureResponse or WebSocket object. Returns subscriber.
unsubscribe(subscriber)

Removes subscriber from self. Returns self.
