CouchDoc Proto







Map with extra methods for easier interaction with CouchDB
 
 
 


after(event, slotName[, target])

Subscribe callback to an after event.
before(event, slotName[, target])

Subscribe callback to an before event.
create()

Creates document in the database. Emits beforeCreate and afterCreate events.
delete()

Deletes document from database. Emits beforeDelete and afterDelete events.
from(map)

Converts Map to CoucDoc.
id()

Returns value of _id property.
listenTo(event, slotName[, target])

Subscribes callback to an event on self instance. If target is provided, its slot slotName will be called.
Returns NotificationListener.
rev()

Returns value of _rev property.
save()

Alias of CouchDoc update().
update()

Saves document to database. Emits beforeUpdate and afterUpdate events.
