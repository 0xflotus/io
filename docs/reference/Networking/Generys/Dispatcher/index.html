Dispatcher Proto







An HTTP request dispatcher, it is responsible for selecting right route depending on the requested path and finalize rendering - activating the right ResponseFormatter.
 
 
 


dispatch(request, response[, candidateAt])

Selects right route and activates it.
formatResponse(controllerResponse, request, response)

Activates ResponseFormatter for controllerResponse.
handleRequest(request, response)

Prepares path, calls dispatch() and formats response.
