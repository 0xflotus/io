FutureResponse Proto







FutureResponse is an class for working with HTTP pooling.
 
 
 


close()

Closes socket and removes self from Generys futureResponses.
closeSocket()

Closes only socket.
finish()

Flushes the data and closes socket.
flush()

Calls Generys prepareData and writes the data to the client.
prepareData()

Converts data to JSON array and empties queue. Returns JSON string.
send(message)

Appends message to the queue.
