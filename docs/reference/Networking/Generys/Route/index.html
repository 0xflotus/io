Route Proto







Route is the core object of the whole routeing system within Generys, but the end-user should never directly use it (wrappers RouteMatch and ResourceMatch should be used instead).
 
 
 


asRegex

Returns RegEx which can be tested agains URLs.
fullNamedCaptures

Returns matches of URL pattern (with ":" or "*" at beginning) as List of Sequences.
interpolate(context)

Interpolate route's pattern. context can be either Map, either Object.

Example: 

  Io> r interpolate({resource: "iceCream", path: "ledo/strawberry"})
  ==> "/iceCream/ledo/strawberry"
mapToPath(path)

Maps Route namedCaptures to values from real path. Returns Map.
Example: 

  Io> Route with("/:resource/*path") mapToPath("/chocolate/swiss/dark") asJson
  ==> {"resource": "chocolate", "path": "swiss/dark"}
namedCaptures

Removes ":" and "*" from strings returned by Route fullNamedCaptures. Returns List.
patternMatches

Extracts :keyword from patterns. Returns RegexMatches object. Example: 

  Io> r := Route with("/:resource/*path")
  Io> r patternMatches all map(string)
  ==> list(":resource", "*path")
respondsTo(path, httpMethod)

Returns true if path matches defined pattern. Otherwise, false.
with(pattern)

Creates a new route and appends it to Generys routes list.
