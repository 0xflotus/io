Router Proto







Router.
 
 
 


DELETE(pattern)

Creates Route which will only respond to HTTP DELETE. Returns RouteMatch.
GET(pattern)

Creates Route which will only respond to HTTP GET. Returns RouteMatch.
POST(pattern)

Creates Route which will only respond to HTTP POST. Returns RouteMatch.
PUT(pattern)

Creates Route which will only respond to HTTP PUT. Returns RouteMatch
connect(pattern)

Creates Route with pattern and returns RouteMatch.
defaultRoutes()

Assings default routes which enables acces to all controllers and their actions.
fileServerRoutes()

Assings route wich will check if a file exists in Generys staticDir which matches request path. If such file exits it will be served to the client. Note: this method has to be called explicitly in routes.io as a last route.
resource(name)

Returns RouteMatch.
