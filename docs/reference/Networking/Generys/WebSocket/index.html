WebSocket Proto







Object for working with WebSocket specification.
 
 
 


channel

Channel to which self is subscribed to (if any).
close()

Closes socket and removes self from Generys webSockets.
handler

WebSocketHandler object whose slots will be called on certain events.
handshake()

Performs WebSocket handshake.
handshaked

true if WebSocket handshake has been completed, false otherwise.
listen()

Internal method which acts on data received by client.
name

Name of the connection. It has to be unique.
origins

List or string of allowed origins of requests. Defaults to "*" (all requests accepted)
patchResponse(response)

Disables Volcano from closing the socket.
requestHeaders

Headers of the request, as given by Volcano.
resource

Path at which connection has been established. (URL path, set automatticaly from Controller createWebSocket to the path of request).
responseHeaders

Headers which will be sent to client during handshake.
send(data)

Sends data to the client. Returns self.
setHandler(handler)

Sets handler slot and handler's socket slot.
socket

Actual Socket with the client.
verifyHeaders

Checks if request contains all headers defined by WebSocket specification. Returns false if not.
verifyOrigin(origin)

Checks if requests origin is allowed.
with(name, httpRequest, httpResponse)

Creates new WebSocket instance.
