SGMLElement Proto







 
 
 


addClassName(className)

Adds class name to caller if absent.
adopt(htmlCode)

Converts htmlCode to SGMLElement and sets caller as its parent.
append(element)

Appends element at the end of the caller element.
asString

Returns a String representation of the tag and all of it's subitems.
attributes

Returns a Map containing the tag's attributes.
elementsBySelector(selector, results)

Returns direct children of current element which are matched by selector. It can be either Sequence or decomposed selector (Map returned by SelectorDecomposer decompose()). results is an optional argument which may contain List of other elements
find(query)

Returns all child elements (List) relative to the caller which match query.
findFirst(query)

Returns first item of result returned by SGMLElement find().
insertAfter(element)

Inserts element after caller in caller's parent.
insertBefore(element)

Inserts element before caller in caller's parent.
isMatchedBySelector(anSelector)

Tests if element matches anSelector.
name

Returns the tag name
next([move])

Returns element after the caller.
positionInParent()

Returns index at which current element is in its parent.
prepend(element)

Appends element at the bottom of the caller element.
prev([move])

Returns element before the caller.
removeClassName(className)

Removes class name from caller.
root

Returns root element.
setAttribute(attributeName, value)

Sets attribute.
setName(aString)

Sets the tag name. Returns self.
subitems

Returns a List containing the tag's subitems.
